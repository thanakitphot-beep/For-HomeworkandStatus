# HomeworkReminder (โปรแกรมบันทึกการบ้าน)

โปรเจกต์นี้เป็นแอปพลิเคชัน Windows Forms สำหรับช่วยนักเรียน/นักศึกษาในการบันทึกรายการการบ้านที่ได้รับมอบหมาย พร้อมทั้งระบุรายวิชาและสถานะของการบ้านนั้นๆ เพื่อช่วยในการติดตามงานไม่ให้ตกหล่น

##  เทคโนโลยีที่ใช้
* **Language:** C#
* **Framework:** .NET Framework / .NET Core (Windows Forms App)
* **IDE:** Visual Studio 2022

##  วิธีดาวน์โหลดและติดตั้ง
1.  ไปที่หน้า Repository นี้
2.  คลิกที่ปุ่ม **Code** สีเขียว และเลือก **Download ZIP**
3.  แตกไฟล์ ZIP ที่ดาวน์โหลดมา
4.  เข้าไปในโฟลเดอร์ และดับเบิ้ลคลิกไฟล์ `HomeworkReminder.sln` เพื่อเปิดใน Visual Studio

##  วิธีรันโปรแกรม
1.  เปิดโปรเจกต์ด้วย Visual Studio
2.  กดปุ่ม **Start** หรือกดแป้นคีย์บอร์ด **F5** เพื่อรันโปรแกรม

##  ส่วนประกอบหน้าจอและการใช้งาน
หน้าจอหลักประกอบด้วยช่องรับข้อมูลดังนี้:

* **HomeworkID:** รหัสของการบ้าน (ห้ามเว้นว่าง)
* **HomeworkTitle:** ชื่อหัวข้อการบ้าน (ห้ามเว้นว่าง)
* **Subject:** เลือกรายวิชา (เช่น Math, Science, English)
* **Status:** เลือกสถานะงาน (Not Started, Doing, Finished)

### ปุ่มคำสั่ง
* **Add:** กดเพื่อบันทึกข้อมูล ระบบจะตรวจสอบข้อมูล หากครบถ้วนจะแสดง MessageBox สรุปรายการ
* **Clear:** ล้างข้อมูลทั้งหมดในช่องกรอก เพื่อเริ่มกรอกใหม่
* **Exit:** ปิดโปรแกรม

##  ตัวอย่างการใช้งาน
1.  กรอกรหัสการบ้านในช่อง *HomeworkID* (เช่น HW-001)
2.  กรอกชื่อการบ้านในช่อง *HomeworkTitle* (เช่น แบบฝึกหัดท้ายบทที่ 1)
3.  เลือกรายวิชา *Subject* และสถานะ *Status*
4.  กดปุ่ม **Add**
5.  โปรแกรมจะแสดงหน้าต่าง (MessageBox) ยืนยันข้อมูลที่กรอกไป
